# Woordenapp Static Web App

Deze repo is ingericht als statische webapp met de app in de root.

## Structuur

- `index.html`: volledige app (HTML, CSS, JS in 1 bestand)
- `404.html`: fallback-pagina
- `images/`: optionele afbeeldingen voor woordjes (`image: "images/bestand.png"`)
- `assets/`: gereserveerd voor losse css/js/img als je later wilt opsplitsen
